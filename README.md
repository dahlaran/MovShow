# MovShow
Android application to search medias (Movies, Series, Mangas) by using TVMazeAPI

Techno : 
- Kotlin
- Binding
- Retrofit (TVMazeAPI)
- RX (for the Repository)
- LiveData (for UI)

Architecture:
- MVVM
- Clean Archi
